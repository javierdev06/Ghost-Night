# Soul Night 🌙

Clon de Soul Knight hecho en HTML/JS/CSS puro.

## Cómo correr
Abre `index.html` en el navegador (o usa Live Server en VS Code).

## Controles
- **WASD / flechas** — mover
- **Mouse** — apuntar
- **Click** — disparar
