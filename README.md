# Ghost Night 

Un roguelike de mazmorras en 2D desarrollado desde cero con HTML, CSS y JavaScript puro.

## Como jugar

Abre `index.html` en el navegador o usa Live Server en VS Code.

## Controles

- **WASD / flechas** — mover
- **Mouse** — apuntar
- **Click** — disparar / atacar
- **ESC** — pausar
- **R** — reiniciar tras Game Over

## Caracteristicas

- 15 niveles con mazmorras generadas aleatoriamente
- 5 enemigos distintos: Slime, Murcielago, Gremlin, Ogro y Manticora (jefe final)
- 9 armas: Pistola, Escopeta, AK-47, Laser, Cohete, Espada, Mazo, Cuchillo y Sarten
- 6 heroes con stats distintos
- 3 skins: Policia, Fantasma y Cebra
- Cofres de plata y dorado con loot
- Niebla de guerra
- Luciernagas decorativas
- Guardado de progreso con localStorage

## Tecnologias

- HTML5 Canvas
- CSS3
- JavaScript ES6+
